# Software-Engineer

## Requisitos
- Python 3.10+
- Git
- Poetry

## Instalação
1. Clone o repositório:
```bash
git clone <URL_DO_REPOSITORIO>.git
cd <PASTA_DO_PROJETO>
```

2. Instale as dependências com o Poetry:
```bash
poetry install
```

3. Ative o ambiente virtual:
```bash
poetry shell
```

## Como rodar

```bash
poetry run python main.py
```
